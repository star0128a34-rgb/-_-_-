수강신청의 불편함을 해소하기 위해 수강신청 필터링 기능을 구현했습니다.

수강신청 페이지는 중복, 연강, 이동 거리 등을 고려해서 강의를 추천해주는 기능이 없었습니다.
또한, 전공, 교양 과목을 한 번에 조회할 수 있는 기능이 없었습니다.

전공, 교양 과목의 정보를 합쳐서 하나의 데이터로 구성했습니다.
신청 과목을 기준으로 중복, 연강, 이동 거리 등을 이진 데이터로 처리했습니다.
선택한 조건에 따라서 적절한 강의만 출력하는 알고리즘을 구현했습니다.
